## Terminal Web Application

### Getting Started

1. `git clone --recursive git@github.com:ravisharmaflyhomes/flyhomes-terminal.git` (Clone the repository recursively)
2. `cd flyhomes-terminal` (Make sure you are at the root of the directory)
3. `yarn install` (Get all the third party libraries)
4. `yarn start` (Run the app on localhost:3000 or pick the port automatically)

Note: Please make sure your node version is >=12.18.3.
